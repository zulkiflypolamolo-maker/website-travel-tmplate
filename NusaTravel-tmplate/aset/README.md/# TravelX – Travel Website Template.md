# TravelX – Travel Website Template

TravelX is a clean and responsive travel website template built with HTML, CSS, and JavaScript.

## Features

- Responsive layout (desktop, tablet, mobile)
- Sticky navbar with mobile menu
- Hero section with background image
- Popular tours card section
- Professional footer

## Folder Structure

assets/
├── css/
├── js/
├── images/
└── icons/

## How to Use

1. Download the template
2. Open the folder
3. Double click `index.html`

## Customization

- Edit text content in `index.html`
- Change styles in `assets/css/style.css`
- Replace images in `assets/images`
